Enter a URL and if the page has Open Graph information you can publish a post wiht it.

== Description ==

The OG image will be downloaded to your site and set as Featured Image of the newly created post.

From [ogp.me](http://ogp.me/):

> *The Open Graph protocol enables any web page to become a rich object in a social graph. For instance, this is used on Facebook to allow any web page to have the same functionality as any other object on Facebook.*

Translations: Português - Español.

== Installation ==

Extract the zip file and upload the contents to the `wp-content/plugins/` directory of your WordPress installation, and then activate the plugin from plugins page. 

= Uninstall =
If you delete the plugin, the entry will be deleted automatically via `unsinstall.php`.
